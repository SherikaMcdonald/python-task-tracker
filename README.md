# Python Task Tracker

## Overview
This project is a simple Python task tracker that allows users to add, view, update, and delete tasks. I built it to show clean Python logic, file handling, and organized problem-solving. The goal was to create a practical tool with a clear structure and easy-to-follow code.

## Features
- Add new tasks
- View current tasks
- Update existing tasks
- Delete completed tasks
- Save tasks in a text file

## Tools Used
- Python
- Visual Studio Code
- GitHub

## Why I Built It
I built this project to strengthen my Python skills and create a clean example of basic software functionality. This project also reflects my interest in writing organized code and building useful tools.

## What I Learned
- How to structure a small Python project
- How to read from and write to files
- How to build menu-based logic
- How to improve code clarity through testing and review

## How to Run
1. Clone the repository
2. Open the project in Visual Studio Code
3. Run `app.py`
4. Follow the menu prompts in the terminal
